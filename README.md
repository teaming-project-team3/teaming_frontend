## FrontEnd 핵심 기능

- `webRTC`
  - 프로젝트 방에서 socket.io를 이용한 webRTC 환경 구축, 화상회의와 채팅 기능
- `socket.io`
  - 실시간 채팅 기능 구현
- `ci/cd`
  - gitHubAction을 이용한 AWS S3 빌드/배포 자동화과정 구축
